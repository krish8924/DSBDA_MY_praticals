# Prompt
Mi DSBDA practical exam sathi prepare karto aahe. 
Mala coding chi basic knowledge aahe pan deep understanding nahi. 

Ata mi ek .ipynb practical file upload karnaar aahe. 
Tujha kaam:

1. Practical cha full aim simple language madhe explain kar.
2. Pratyek code line:
   - kay karte
   - ka use keli
   - output var kasa effect hoto
   - he very simple Marathi + easy English mix madhe explain kar.
3. Jithe library/function/term use keli aahe tithe:
   - meaning
   - use
   - syntax
   - alternative function/code
   - short oral answer
   explain kar.
4. Variables, loops, conditions, dataframe operations, graphs, ML steps sagla basics pasun explain kar.
5. Oral exam sathi:
   - possible viva questions
   - short simple answers
   - teacher la impress kartil ase points
   de.
6. Important code lines highlight kar ani sang:
   - “hi line viva madhe vicharu shaktat”.
7. Errors aale tar:
   - ka yetat
   - kase solve karayche
   simple madhe explain kar.
8. Output interpretation pan explain kar.
9. Theory + practical connection pan explain kar.
10. End la:
   - full practical cha 2 minute revision
   - important keywords
   - shortcut memory tricks
   de.

Rules:
- Khup simple language use kar.
- Beginner student la shikavtoy asa explain kar.
- Unnecessary technical words avoid kar.
- Each section heading ne organize kar.
- Marathi + easy English mix use kar.
- Long paragraphs nako.
- Tables ani bullet points use kar.

Jar ekach practical madhun multiple concepts astil tar separate sections madhe explain kar.
